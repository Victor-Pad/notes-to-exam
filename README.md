# notes-to-exam
Personal app to improve the self-studying experience. Takes notes, reinterprets them and outputs an exam in PDF format.

# Motivation:
As a big self-learner, I have come to realize just how tough it can be to take it upon yourself to study academic topics. Finding structured content can be hard, and no one, but youself, can make you accountable. 

So I came up with the idea of building an app that aims to make the self-studying process easier. 

By creating exams based on your notes, you could test yourself more effectively and have an experience closer to a real university setting.
